Publish to Forge

Share an app you built with your team. Three ways to do it.

Paste your HTML

Single-file app. Should include all CSS/JS inline (or via CDN).

Tip: drag-drop an .html file directly into this box.

Upload a zip

Folder must contain index.html. Forge serves it as a static bundle.

Deploy from a GitHub repo

Forge will clone the repo, look for index.html or forge.yaml, and publish it. Pushes to main auto-redeploy if you wire the webhook.

Public repos work out of the box. Private repos need a deploy token (set up in Settings).
